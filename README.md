# react-native-
react-native+react-navigation+ant-design/react-native实现一个简单的APP界面

使用了react-navigation实现底部的tab，切换tab页面跳转的功能，使用了ant-designUI组件库

### 拉代码
```
git clone git@github.com:zhoujin123/react-native-.git
```
### 安装依赖
```
yarn 或者 npm install
// ios还需要执行如下操作
cd ios && pod install
```
### 运行
```
yarn ios 或 yarn android
```
